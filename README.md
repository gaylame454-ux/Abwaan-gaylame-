# Website-ka Suugaanta Soomaaliyeed

Website-kan wuxuu u gaar yahay suugaanta Soomaaliyeed, gaar ahaan gabayada, heesaha iyo hal-abuurka dhaqanka.

## Waxyaabaha ku jira
- Gabayo Soomaaliyeed
- Abwaanno caan ah
- Taariikh suugaaneed iwm

## Ujeeddo
In la kaydiyo lana faafiyo suugaanta Soomaaliyeed.

## Sameeye
Arday jecel suugaanta iyo dhaqanka Soomaaliyeed.
